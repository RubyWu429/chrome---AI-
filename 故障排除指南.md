# AI摘要助手 - 故障排除指南

## 🔍 问题诊断步骤

### 第一步：检查扩展是否正确安装

1. **打开Chrome扩展页面**
   - 在地址栏输入：`chrome://extensions/`
   - 查看扩展列表中是否有"AI摘要助手"

2. **检查扩展状态**
   - ✅ 扩展应该显示为"已启用"状态
   - ✅ 扩展图标应该出现在工具栏中
   - ❌ 如果显示"错误"，点击"重新加载"

3. **检查开发者模式**
   - 确保右上角的"开发者模式"开关已开启

### 第二步：检查右键菜单

1. **测试右键菜单**
   - 在任意网页上选择一些文本
   - 右键点击选中的文本
   - 查看右键菜单中是否有"自动摘要并添加到飞书文档"选项

2. **如果没有右键菜单选项**
   - 重新加载扩展：在扩展页面点击"重新加载"
   - 重启Chrome浏览器
   - 检查扩展权限是否正确

### 第三步：检查API配置

1. **打开扩展设置**
   - 点击扩展图标
   - 检查OpenAI API密钥是否已配置

2. **验证API密钥**
   - 确保API密钥格式正确（以`sk-`开头）
   - 确保API密钥有效且有余额

### 第四步：查看控制台日志

1. **打开开发者工具**
   - 按F12或右键选择"检查"
   - 切换到"控制台"标签

2. **查看扩展日志**
   - 在控制台中查找以"🚀"、"📝"、"🤖"等开头的日志
   - 这些是扩展的调试信息

3. **查看错误信息**
   - 查找红色的错误信息
   - 特别注意API调用相关的错误

## 🛠️ 常见问题及解决方案

### 问题1：右键菜单没有选项

**症状**：右键点击选中的文本时，没有看到"自动摘要并添加到飞书文档"选项

**解决方案**：
1. 重新加载扩展
2. 重启Chrome浏览器
3. 检查扩展权限

**检查步骤**：
```javascript
// 在控制台中运行以下代码检查右键菜单
chrome.contextMenus.getAll((menus) => {
    console.log('右键菜单项:', menus);
});
```

### 问题2：点击右键菜单没有反应

**症状**：右键菜单选项存在，但点击后没有任何反应

**解决方案**：
1. 检查控制台是否有错误信息
2. 确认已配置OpenAI API密钥
3. 检查网络连接

**调试步骤**：
1. 打开控制台
2. 选择文本并点击右键菜单
3. 查看控制台输出的调试信息

### 问题3：API调用失败

**症状**：显示"OpenAI API错误"或"处理失败"

**可能原因**：
- API密钥无效或已过期
- API密钥余额不足
- 网络连接问题
- API请求格式错误

**解决方案**：
1. 检查API密钥是否正确
2. 登录OpenAI账户检查余额
3. 检查网络连接
4. 尝试重新生成API密钥

### 问题4：没有复制到剪贴板

**症状**：摘要生成成功，但没有自动复制到剪贴板

**可能原因**：
- 浏览器权限问题
- 剪贴板API不可用
- 安全策略限制

**解决方案**：
1. 检查浏览器是否允许剪贴板访问
2. 手动复制显示的摘要内容
3. 使用备用方案（直接显示内容）

### 问题5：没有打开飞书文档页面

**症状**：摘要生成成功，但没有自动打开飞书文档页面

**可能原因**：
- 弹出窗口被阻止
- 网络连接问题
- 飞书链接无效

**解决方案**：
1. 检查浏览器是否阻止弹出窗口
2. 手动打开飞书文档链接
3. 检查网络连接

## 🔧 高级调试

### 使用简化版本测试

如果原版本有问题，可以尝试使用简化版本：

1. **替换background.js**
   - 将`background-simple.js`重命名为`background.js`
   - 重新加载扩展

2. **简化版本特点**
   - 更详细的调试信息
   - 简化的错误处理
   - 更可靠的备用方案

### 手动测试API

在控制台中手动测试OpenAI API：

```javascript
// 测试API调用
async function testAPI() {
    const apiKey = 'your-api-key-here';
    const response = await fetch("https://api.openai.com/v1/chat/completions", {
        method: "POST",
        headers: {
            "Content-Type": "application/json",
            "Authorization": `Bearer ${apiKey}`
        },
        body: JSON.stringify({
            model: "gpt-3.5-turbo",
            messages: [
                {
                    role: "user",
                    content: "请简单回复'测试成功'"
                }
            ],
            max_tokens: 50
        })
    });
    
    const data = await response.json();
    console.log('API测试结果:', data);
}

testAPI();
```

### 检查扩展权限

```javascript
chrome.permissions.getAll((permissions) => {
    console.log('扩展权限:', permissions);
});
```
