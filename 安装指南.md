# AI摘要助手 - 安装指南

## 🚀 快速开始

### 第一步：准备API密钥

1. **获取OpenAI API密钥**
   - 访问 [OpenAI官网](https://platform.openai.com/)
   - 注册或登录账号
   - 进入 [API Keys页面](https://platform.openai.com/api-keys)
   - 点击"Create new secret key"创建新的API密钥
   - 复制并保存这个密钥（注意：密钥只显示一次！）

2. **获取飞书应用配置（可选）**
   - 访问 [飞书开放平台](https://open.feishu.cn/)
   - 创建新应用或使用现有应用
   - 在应用详情页面获取App ID和App Secret
   - 配置应用权限（需要表格读写权限）

### 第二步：安装Chrome扩展

1. **下载扩展文件**
   - 确保所有文件都在同一个文件夹中
   - 文件结构应该如下：
   ```
   AI摘要助手/
   ├── manifest.json
   ├── background.js
   ├── content.js
   ├── popup.html
   ├── popup.js
   ├── feishu-api.js
   ├── README.md
   ├── 安装指南.md
   └── icons/
       ├── icon16.png
       ├── icon48.png
       └── icon128.png
   ```

2. **在Chrome中加载扩展**
   - 打开Chrome浏览器
   - 在地址栏输入：`chrome://extensions/`
   - 开启右上角的"开发者模式"开关
   - 点击"加载已解压的扩展程序"
   - 选择包含所有扩展文件的文件夹
   - 扩展应该出现在扩展列表中

### 第三步：配置扩展

1. **打开扩展设置**
   - 点击Chrome工具栏中的扩展图标
   - 或者右键点击扩展图标选择"选项"

2. **输入配置信息**
   - **OpenAI API密钥**：粘贴你获取的API密钥
   - **飞书应用App ID**：输入飞书应用的App ID（可选）
   - **飞书应用App Secret**：输入飞书应用的App Secret（可选）
   - 点击"保存设置"

## 📖 使用方法

### 基本使用流程

1. **选择文本内容**
   - 打开任意网页（如微信公众号文章）
   - 使用Ctrl+A（Mac上是Cmd+A）全选页面内容
   - 或者手动选择要摘要的文本段落

2. **生成摘要**
   - 右键点击选中的文本
   - 在右键菜单中选择"自动摘要并添加到飞书文档"
   - 等待AI生成摘要（通常需要几秒钟）

3. **查看结果**
   - 如果配置了飞书API，摘要会自动添加到飞书表格
   - 如果没有配置飞书API，会显示摘要内容并打开飞书文档页面
   - 可以手动复制摘要内容到飞书表格

### 飞书表格格式

摘要会自动添加到以下格式的表格中：

| 摘要 | 链接 | 标题 | 时间 |
|------|------|------|------|
| AI生成的摘要内容 | 原文章链接 | 页面标题 | 添加时间 |

## ⚙️ 高级配置

### 飞书API配置详解

如果要实现完全自动化的功能，需要配置飞书API：

1. **创建飞书应用**
   - 登录[飞书开放平台](https://open.feishu.cn/)
   - 点击"创建应用"
   - 填写应用名称和描述

2. **配置应用权限**
   - 在应用详情页面，进入"权限管理"
   - 添加以下权限：
     - `bitable:app`（多维表格应用权限）
     - `bitable:app:readonly`（多维表格应用只读权限）
     - `bitable:table`（多维表格权限）
     - `bitable:table:readonly`（多维表格只读权限）

3. **获取应用凭证**
   - 在"应用凭证"页面获取App ID和App Secret
   - 将这些信息填入扩展设置

4. **发布应用**
   - 在"版本管理与发布"页面创建版本
   - 提交审核并发布应用

### 自定义摘要长度

可以在`background.js`文件中修改摘要长度：

```javascript
// 在generateSummary函数中修改max_tokens参数
max_tokens: 500, // 增加这个数值可以获得更长的摘要
```

## 🔧 故障排除

### 常见问题及解决方案

1. **扩展不显示右键菜单**
   - 检查扩展是否正确安装
   - 尝试重新加载扩展
   - 确保在正确的网页上使用

2. **API密钥错误**
   - 检查OpenAI API密钥是否正确
   - 确认API密钥有足够的余额
   - 检查网络连接

3. **飞书API调用失败**
   - 检查App ID和App Secret是否正确
   - 确认应用权限配置正确
   - 检查目标表格是否存在且有访问权限

4. **摘要生成失败**
   - 检查选中的文本内容是否有效
   - 确认网络连接正常
   - 查看浏览器控制台的错误信息

### 调试模式

1. **查看扩展日志**
   - 右键点击扩展图标
   - 选择"检查弹出内容"
   - 查看控制台输出

2. **查看后台脚本日志**
   - 访问`chrome://extensions/`
   - 找到扩展，点击"详情"
   - 点击"检查视图：背景页"

## 💡 使用技巧

1. **优化摘要质量**
   - 选择包含主要内容的文本段落
   - 避免选择广告、导航等无关内容
   - 对于长文章，可以分段摘要

2. **提高效率**
   - 使用快捷键Ctrl+A快速全选
   - 将常用的网页添加到书签
   - 定期清理飞书表格中的重复内容

3. **节省API费用**
   - 合理控制摘要频率
   - 对于相似内容可以复用摘要
   - 监控OpenAI API使用量

4. 联系开发者获取技术支持
